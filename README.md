# BankAccount class
`BankAccount` class нь дараах үйлдлүүдийг гүйцэтгэнэ:
  - Дансанд мөнгө хийх (deposit)
  - Данснаас мөнгө авах (withdraw)
  - Үлдэгдэл шалгах (getBalance)

### Лог :
- TRACE — метод руу орох/гарах (entry/exit)
- DEBUG — өмнөх ба дараах үлдэгдэл
- INFO — амжилттай гүйлгээ
- WARN — сөрөг эсвэл буруу хэмжээ оруулсан
- ERROR — үлдэгдэлээс их мөнгө авах гэж оролдох
- FATAL — системийн критик нөхцөлийн жишээ (жишээ: config алдаатай гэж заах)

# Customer class
`Customer` class нь хэрэглэгчийн мэдээллийг хадгална.
  - Нэр болон имэйл хадгалах
  - Өгөгдлийн шалгалт хийх

### Сайжруулалт:
- Null хамгаалалт хийсэн
- Имэйл форматын шалгалт нэмсэн
- Нууц мэдээллийг масклах боломжтой болсон

## ⚙️ log4j2.xml тохиргоо
Log4j2 нь тусгай тохиргооны файл ашиглана
  - Console болон File appender ашигласан
  - Pattern layout-д дараах мэдээлэл багтсан:
    - Огноо     - Thread    - Class (%c)
    - Line number     - Message

tail -f нь log файлыг шинэчлэгдэж байгааг тасралтгүй харуулдаг тул систем 
ажиллаж байх үед гарч буй алдаа, үйл явцыг шууд ажиглах боломжтой. 
cat болон less нь зөвхөн тухайн агшны статик мэдээллийг харуулдаг.
